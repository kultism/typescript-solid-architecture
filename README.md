# NODEJS ARCHITECTURE WITH SOLID PRINCIPLES 

## Running
### 1. Install Package
- npm install
- npm install -D @types/express @types/mongoose

### 2. Testing
npm run test

### 3. Run in development mode
npm run dev

### 4. Build
npm run build

### 5. Run in production mode
npm run start
